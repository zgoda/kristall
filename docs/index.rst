Kristall
========

Kristall is minimalist Python WSGI framework for writing APIs and backends. It
uses `Werkzeug <https://palletsprojects.com/p/werkzeug/>`_ as WSGI toolbox and
itself is only very thin wrapper over what Werkzeug provides.
